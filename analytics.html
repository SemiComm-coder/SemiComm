GA4 status: checking...

This setup uses Google Analytics 4 for tracking and an optional Looker Studio embed for a single internal dashboard view.

Setup checklist

  1. Create a Google Analytics 4 property and add your website stream.
  2. Set your GA measurement ID in script.js by replacing G-XXXXXXXXXX.
  3. Create a Looker Studio report from GA4 with these widgets: total users, top pages, users by country, and traffic source/medium.
  4. Set the Looker Studio embed URL in script.js by replacing LOOKER_STUDIO_EMBED_URL.
